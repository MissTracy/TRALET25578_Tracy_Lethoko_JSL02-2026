# Project: Task Input and Status Validation System


## Overview
This project involves implementing a JavaScript-based task entry system where users can input two tasks with complete details, ensuring data consistency and validation. The system will guide users through entering valid task titles, descriptions, and statuses, enforcing correct formatting while maintaining code clarity and maintainability.(project is demonstation of using simple loops for tasks)


## Technologies Used

Language: JavaScript ,HTML & CSS
Environment: Web Browser 

## Features

-Task Capture: Collects titles and detailed descriptions for multiple tasks.

-Status Validation : Uses a persistent logic loop to ensure users can only enter 'todo', 'doing', or 'done'.

-Input: Automatically converts status inputs to lowercase to prevent case-sensitive errors.

-Progress Feedback: Provides specific console feedback based on whether tasks are completed or still in progress.
